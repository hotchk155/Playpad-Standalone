USBHost Generic Device Driver and Application Sample Readme
-----------------------------------------------------------

Example layered driver for a USB device. Provides a read, write and IOCTL interface to USB device.
Connect to the device using the attach IOCTL function then read and write using the vos_dev_read and vos_dev_write functions.
